Last month, Adobe released AEM 6.2, the latest iteration of Adobe Experience Manager, Adobe's flagship Web Experience Management product. This new release comes with some exciting new features including:

  • Template Editor
  • Smart Tags
  • Asset Insights
  • AEM Mobile
  • Community Insights & Scoring

One of the most exiciting features for me is the latest and greatest integration with Adobe Target for Personalization and Optimization. This new itegration provides a much deeper integration between the authoring and content management features of Adobe Experience Manager and the powerful personalization and optimization features of Adobe Target.

While at Adobe Summit, I attended the Lab "Integrating Experience Manager with Adobe Analytics, Target and DTM", this lab showed the new integration between AEM and Target. In only one hour, we were able to set up all four products and integrate them together into a cohesive solution to personalize content.
